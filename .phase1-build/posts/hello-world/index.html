TL;DR

Tóm tắt trước khi đọc

  • Ví dụ khởi tạo AES cipher trong Go với gói crypto/aes.
  • Key phải đúng độ dài chuẩn (16/24/32 bytes) để tránh lỗi invalid key size.
  • Đây là bước nền trước khi xây Encrypt/Decrypt hoàn chỉnh cho project.

Dưới đây là cấu trúc khởi tạo một luồng mã hóa bảo mật cơ bản:

package main

import (
	"crypto/aes"
	"crypto/cipher"
	"fmt"
)

func main() {
	key := []byte("AES256Key-32Characters1234567890")
	block, err := aes.NewCipher(key)
	if err != nil {
		panic(err.Error())
	}
	fmt.Printf("Block size: %d bytes\n", block.BlockSize())
}
